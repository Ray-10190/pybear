什麼是 Python？

Python 是一種程式語言，就像你用中文或英文和別人說話一樣，程式語言可以讓你跟電腦說話，告訴它要做什麼。它的語法簡單、好讀，很多初學者都喜歡用它。

🔤

簡單的指令：print()

用 print() 可以讓電腦在螢幕上顯示東西。

print("你好，Python！")

點右邊範例按鈕可以看到模擬結果。

🔁

循環（做很多次的事情）

用 for 迴圈可以讓電腦重複做事情。

for i in range(3): print("嗨！", i)

常見小詞彙

  • 變數（variable）：像是一個盒子，放東西（數字、文字）。
  • 函式（function）：做一件事情的指令集合，像是玩具的組裝說明書。
  • 迴圈（loop）：重複做同一件事。
  • 條件（if）：遇到不同情況做不同事，例如「如果下雨就帶傘」。
📚

學習小秘訣

  1. 多練習：每次只學一件小事。
  2. 像說故事一樣寫程式，想像你在和電腦對話。
  3. 錯誤是好事，會讓你學到更多——別怕試錯。
